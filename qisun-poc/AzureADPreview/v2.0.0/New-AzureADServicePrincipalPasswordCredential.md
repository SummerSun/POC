# New-AzureADServicePrincipalPasswordCredential

## SYNOPSIS

## SYNTAX

```
 [-StartDate <DateTime>] [-EndDate <DateTime>] [-Value <String>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```
