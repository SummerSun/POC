# Get-AzureRMManagedLocation
## SYNOPSIS
Gets all the regions registered in the cloud.
This cmdlet is applicable only to Microsoft Azure Stack Environments

## SYNTAX

```
Get-AzureRMManagedLocation [-Name <String>] [-SubscriptionId <Guid>] [-AdminUri <Uri>] [-Token <String>]
 [-ApiVersion <String>] [-DisableCertificateValidation] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>]
```
