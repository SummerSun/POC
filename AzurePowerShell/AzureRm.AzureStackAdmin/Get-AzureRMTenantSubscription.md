# Get-AzureRMTenantSubscription
## SYNOPSIS
Gets a list of Subscriptions that a user has access to..
This cmdlet is applicable only to Microsoft Azure Stack Environments

## SYNTAX

```
Get-AzureRMTenantSubscription [-AdminUri <Uri>] [-Token <String>] [-ApiVersion <String>]
 [-DisableCertificateValidation] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```
